# Méthodologie de l'Observatoire Holia

Ce dossier décrit comment l'Observatoire Holia produit, structure et publie ses données ouvertes. Il s'adresse aux chercheurs, journalistes, décideurs publics et réutilisateurs de données qui souhaitent comprendre le périmètre et les limites des publications.

## Principes directeurs

1. **Transparence** : chaque étude indique son objet, sa population, sa date d'arrêté et ses limites.
2. **Reproductibilité** : les jeux de données CSV sont versionnés par édition (`2026-05`, `2026-06`, etc.).
3. **Anonymisation** : aucune donnée personnelle identifiable n'est publiée dans les agrégats.
4. **Périmètre explicite** : les chiffres décrivent le référentiel Holia, pas le marché français dans son ensemble.
5. **Absence de valeur médicale** : l'Observatoire ne produit ni diagnostic, ni recommandation thérapeutique.
